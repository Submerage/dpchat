# 欢迎使用本地文档阅读器

这是一个示例文本文件，放在resources文件夹中。

## 功能特点
- 无需网络连接（一旦加载）
- 支持多种文档格式：TXT, DOCX, PDF, MD, HTML
- 简洁的阅读界面
- 响应式设计，适配各种设备
- 完全在GitHub Pages上免费部署

## 如何在GitHub Pages使用
1. 将您的文档放入resources文件夹
2. 提交并推送到GitHub仓库
3. 在GitHub仓库设置中启用GitHub Pages
4. 访问生成的GitHub Pages链接
5. 从左侧列表选择文档开始阅读

## 技术栈
- HTML5
- CSS3
- JavaScript
- Mammoth.js (用于解析DOCX)
- PDF.js (用于解析PDF)
- GitHub Pages (免费托管)

注意：所有文档必须预先提交到resources文件夹，此应用无法访问您本地计算机上的其他文件。

感谢使用本地文档阅读器！
